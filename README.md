The Old Phone Price Prediction System is a machine learningâ€“based application designed to estimate the resale price of used smartphones based on key features such as brand, model, age, RAM, storage, battery capacity, camera specifications, and physical condition. The final model can be integrated into a web application where users input phone specifications and receive an estimated resale price instantly. This project demonstrates practical skills in data preprocessing, regression modeling, model evaluation, and end-to-end machine learning pipeline development.
